This project uses Maven v1.1.b2 for the build

Running "maven multiproject:site" from top level of source 
directory will generate a Maven project web site
in the target/docs directory.

Documentation

External Dependencies:
	JBoss 4.0.2/4.0.3SP1
	JDK 1.5.x
	MySql 4.x (InnoDB)

Windows distribution structure:
        /document/install.pdf  
	/Setup.exe	  (Native Windows Installer for all components)
	/DMSetup.exe  (Device Manager Only native Installer - See install guide)
	


UNIX distribution structure:
	/document/install.pdf
	/install   (contains built image that installer uses)
